> **关于 trailingSlash：** 当前 `next.config.ts` 无 `trailingSlash` 设置（默认 false），产物为 `tools/claude-code.html` 而非 `tools/claude-code/index.html`。EdgeOne Pages 通常支持两种格式；若部署后出现路由 404，可在 `next.config.ts` 添加 `trailingSlash: true` 并重新构建。
