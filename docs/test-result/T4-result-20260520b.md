### TC-RUN2：`yc.main()` 首页失败抛 `RuntimeError`，不调用 `sys.exit`

**验证方式：** mock `fetch_json` 返回 `None`，捕获异常类型

**验证代码：**
```python
import sys
sys.path.insert(0, 'scripts/crawler')
import _common
_common.fetch_json = lambda *a, **k: None

from yc import main
try:
    main(dry_run=True)
except RuntimeError as e:
    print('PASS - RuntimeError caught:', e)
except SystemExit:
    print('FAIL - sys.exit called')
```

**实测输出：**
```
PASS - RuntimeError caught: YC API 第一页抓取失败，可能是网络问题或 API 暂时不可用
```

**结论：** ✅ PASS — `run.py` 的 `except Exception` 可正常捕获，不再通过 `SystemExit` 协议通信
