# AI 驱动的全球热门 AI 项目数据平台和推荐项目 

**版本：V1.1 | 日期：2026 年 5 月 13 日**

**一句话定义：基于成熟开源组件，快速搭建融合 Toolfy 榜单与闲鱼交易逻辑的 AI 项目数据平台，45 天完成 MVP 交付并验证业务闭环。**

---

## 一、产品定位

### 1.1 产品定位

本系统是 "数据采集自动化 + 榜单展示 + 撮合交易" 三位一体的 AI 项目服务平台，以 Toolfy 的 "全球 AI 项目数据采集 + 榜单展示" 为基础，融合闲鱼 "商品发布 + 即时咨询 + 撮合交易" 的轻量化逻辑，聚焦 YC、斯坦福、哈佛等机构孵化的新兴 AI 创业项目，打造差异化的实时运营数据优势，服务企业 / 个人用户的工具筛选与项目交易需求。

### 1.2 核心原则

- **绝对复用原则**：能用开源、能用现成组件、能用第三方接口绝不从零开发，只写业务特有的逻辑
- **功能减法原则**：只保留能跑通业务闭环的最小要素，砍掉所有美化、高级功能、复杂运营
- **分段迭代原则**：45 天拆分为 6 个周版本，每周一个可演示、可自测、可小范围灰度的版本
- **先验证后优化原则**：先验证 "采→洗→展→发→聊" 业务闭环，再调 UI、算法、评分等体验
- **AI 驱动开发原则**：人定意图和标准，AI 执行产出，人 review 和调整，最大化开发效率
- **职责清晰原则**：严格区分前后端职责，建立统一的 API 规范和文档体系，避免交叉混乱

## 二、技术选型：全部用成熟第三方

### 2.1 技术栈总览

| 层              | 选型                              | 为什么选它                                          | 替代方案                   |
| :-------------- | :-------------------------------- | :-------------------------------------------------- | :------------------------- |
| **爬虫框架**    | **Crawl4AI**（12k stars）         | AI 驱动的智能爬虫，支持动态页面、反爬绕过，开箱即用 | Scrapy + Playwright        |
| **数据清洗**    | **LangChain + Claude 3.5 Sonnet** | 语义理解能力强，适合非结构化数据处理                | LlamaIndex + Qwen2.5-14B   |
| **后端框架**    | **FastAPI**                       | 高性能、自动生成 OpenAPI 文档，Python 生态友好      | Django REST Framework      |
| **前端框架**    | **Vue 3 + Ant Design Vue 4.x**    | 成熟的企业级组件库，二开友好，有大量现成榜单模板    | React + Ant Design         |
| **后台管理**    | **Ant Design Pro Vue**            | 开箱即用的后台管理系统，支持数据 CRUD、权限管理     | Element Plus Admin         |
| **数据库**      | **PostgreSQL 16 + PgVector**      | 支持关系型数据和向量存储，适合项目数据和 RAG 场景   | MySQL + Milvus             |
| **定时任务**    | **Celery 5.x + Redis 7.x**        | 成熟的分布式任务队列，支持定时执行和任务重试        | APScheduler                |
| **即时聊天**    | **环信 IM 4.x**                   | 成熟的第三方 IM 组件，一键嵌入，支持私聊、群聊      | 融云 IM                    |
| **登录 / 短信** | **阿里云短信服务**                | 稳定可靠，接口简单                                  | 腾讯云短信服务             |
| **部署**        | **Docker Compose 2.x**            | 所有组件一个 compose 文件拉起，一键部署             | Kubernetes（MVP 阶段不用） |
| **监控**        | **Uptime Kuma**（开源）           | 轻量级服务监控，Docker 部署                         | Prometheus + Grafana       |
| **日志**        | **ELK Stack 8.x**（简化版）       | 集中式日志管理，便于排查问题                        | 本地日志文件               |
| **API 文档**    | **FastAPI 自动生成 + Swagger UI** | 自动生成交互式 API 文档，无需手动编写               | Postman 文档               |
